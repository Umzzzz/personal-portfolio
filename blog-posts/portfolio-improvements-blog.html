Improvements Of Portfolio Website

Improvements have been made from the feedback I have received from fellow students and from the live marking session done by my tutor few weeks ago.

There were some minor issues with the first version of the site. First of all I changed the logo to match with the colour scheme of the navigation, which is black background with gold hover. So this made me change the Favicon to match the brand identity of Umair Yaquoob. To do this I used an online tool called Favicon Generator.

“Google typically displays the first 50-60 characters of a title tag, or as many characters as will fit into a 512-pixel display. If you keep your titles under 55 characters, you can expect at least 95% of your titles to display properly.” (MOZ, no date: online)

So I decided to do what MOZ stated by reducing the title tag to less than 55 characters, so that Google can display the titles properly in the browsers tab area, search engine results and external websites.(sharing on social media sites, they use the title tag as its link anchor text)

Meta description was too short on the main page of the site, so I increased the length of the Meta description to sell myself.

In some places, the Alt texts for images were redundant for example I used ‘Image of', which is no use to impaired users. So I approach this issue by making the Alt text more meaningful to the relevant images. On the other hand, related to the navigation in responsive mode, the grey colour text was not properly visible in mobile view. So I changed the colour of the text to white, so that it was more clearly visible for the users to read.

In responsive mode, especially the mobile column width, two section on the main page ‘About’ and ‘Blog’, the written content was to narrow. To resolve these issues, I changed the padding to 20px which fixed the problems and allowed the content to be more readable.

Furthermore, this time round I managed to get the off click outside navigation effect to work with advice from Jacob Gray at Stack Overflow.

While in responsive mode, when the user clicks on the toggle menu, the menu slides down, imagine the user wants to click outside the navigation menu, this effect will make this happen and the navigation menu will disappear automatically, instead of clicking the toggle button again.

On the other hand, to make the website load even faster, I managed to minify all the JavaScript files into one Script file. This was done using an online tool called JavaScript Compression Tool. Also I minified the CSS, using an online tool called minify your CSS.

Final improvement that I have made to the site was replacing the home section icons and social media icons, using the popular web icons platform Font Awesome. Also I have changed the footer text from Umz Designs to Umair Yaquoob to match with the initials in the logo.
